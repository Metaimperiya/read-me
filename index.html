Технологии, используемые в проекте

📋 Полный список

КомпонентТехнология
FrontendHTML5, CSS3, Vanilla JavaScript
BackendFirebase (Realtime Database, Authentication, Storage)
АутентификацияFirebase Auth (Google + Anonymous)
Хранение данныхFirebase Realtime Database
Хранение файловFirebase Storage (аватарки)

⚠️ Важные замечания

  • Ключи Firebase должны храниться в .env (переменные окружения)
  • На Vercel нужно добавить переменные: FIREBASE_API_KEY, FIREBASE_AUTH_DOMAIN, FIREBASE_DATABASE_URL, FIREBASE_PROJECT_ID, FIREBASE_STORAGE_BUCKET, FIREBASE_APP_ID
  • Правила БД — в database.rules.json
  • Правила Storage — в storage.rules

📄 sw.js — Service Worker

  • Кеширование — загружает сайт в кеш при первом открытии
  • Офлайн-режим — сайт работает без интернета
  • Быстрая загрузка — повторные открытия мгновенные

Для регистрации SW добавь в index.html:

if ('serviceWorker' in navigator) {
    navigator.serviceWorker.register('/sw.js');
}
